# Back-end-Development-Capstone

## 🎸 Aplicación Principal - Sitio Web de Banda de Rock

Este es el proyecto final del curso IBM Back-End Development. Es una aplicación web completa desarrollada con Django que sirve como sitio web para una banda de rock, integrando servicios de canciones, fotos y conciertos.

---

## 🚀 Configuración del Entorno

### Requisitos Previos
- **Python 3.9.x** (versión específica: 3.9.18 o superior)
- **Git** (para control de versiones)
- **pip** (gestor de paquetes de Python)
- **SQLite** (base de datos por defecto)
